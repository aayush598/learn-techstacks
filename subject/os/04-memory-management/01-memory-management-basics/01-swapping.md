# Swapping

## Logical vs Physical Address Space

- **Logical address (virtual):** generated by CPU, used by process
- **Physical address:** actual location in RAM (main memory)
- Programs deal with logical addresses; OS/hardware maps to physical

## MMU (Memory Management Unit)

- Hardware device that translates logical → physical addresses at runtime
- CPU sends logical address → MMU maps it → physical address sent to memory
- Base/limit registers in MMU for relocation and bounds checking
- Every memory reference goes through MMU (no software overhead)

## Swapping Concept

- **Swap out:** process moved from main memory to **backing store** (disk)
- **Swap in:** process brought back from backing store to memory
- Purpose: free up RAM for other processes (multiprogramming)
- Backing store: fast disk partition (swap partition) large enough for all processes
- Process must be **idle** (not pending I/O) during swap

## Swapping Overhead Calculation

- Swapping time dominated by **transfer time** (seek + rotational latency are small)
- `swap_time = swap_in + swap_out = 2 × (process_size / transfer_rate)`

| Process Size | Transfer Rate | Swap Time |
|---|---|---|
| 100 MB | 50 MB/s | 4 seconds |
| 1 GB | 200 MB/s | 10 seconds |
| 10 MB | 50 MB/s | 0.4 seconds |

**Example:** 100 MB process, 50 MB/s disk
- `swap_out = 100/50 = 2s`, `swap_in = 2s`, total context switch including swap ≈ **4s**

## Context Switch Time with Swapping

- Swap processes: time to swap out current + swap in next
- **Dispatch latency** = time to make process ready after swap
- Directly proportional to process size
- Too slow for modern interactive systems

## Modern Approach: Lazy Swapper

- **Swapper (scheduler):** swaps entire processes (old approach)
- **Lazy swapper:** never swaps a page into memory unless needed (demand paging)
- Modern OS **swap pages**, not whole processes
- Linux: **kswapd** daemon manages page-level swapping
- **Swap space** used as extended memory, not full-process swap
- Key insight: most processes don't need all their pages resident
