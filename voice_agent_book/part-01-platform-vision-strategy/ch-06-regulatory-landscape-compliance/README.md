## Key Takeaways

- US: Two-party consent states (11 states) require all-party consent for recording
- EU: GDPR requires explicit consent, data portability, right to erasure
- Healthcare: HIPAA BAAs required with all vendors processing PHI
- Finance: PCI DSS compliance required for any payment processing
- Compliance is a competitive advantage, especially for enterprise sales
