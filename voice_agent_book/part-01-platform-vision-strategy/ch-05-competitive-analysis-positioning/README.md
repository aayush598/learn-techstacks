## Key Takeaways

- Competitors are well-funded but mostly closed-source
- Open-source + white-label is a unique differentiator
- SMB market is underserved by current pure-play voice AI platforms
- Key moats: compliance certifications, community, integration ecosystem
- Big tech (Google, Microsoft, Amazon) is the biggest long-term threat
