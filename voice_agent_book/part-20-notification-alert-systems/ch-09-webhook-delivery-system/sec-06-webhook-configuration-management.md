# Section 06: Webhook Configuration & Management

## Overview

Consumers configure webhook endpoints through the developer portal, selecting which event types to receive, managing secrets, and setting rate limits. The configuration API supports full CRUD on endpoints, event type subscription management, and secret rotation.

## Design Decisions

- **Self-Service Configuration**: Consumers manage endpoints via UI/API
- **Event Type Selectors**: Granular subscription control
- **Test Endpoint**: Send test event to validate configuration
- **Secret Auto-Generation**: Strong secrets generated by default

## Implementation Approach

```typescript
interface WebhookEndpoint {
  id: string;
  tenantId: string;
  url: string;
  description?: string;
  secret: string;
  secondarySecret?: string;
  subscribedEvents: string[];
  enabled: boolean;
  retryConfig: {
    maxAttempts: number;
    baseDelayMs: number;
  };
  rateLimit: {
    requestsPerMinute: number;
    currentWindow: number; // timestamp
    windowCount: number;
  };
  headers?: Record<string, string>;
  createdAt: Date;
  updatedAt: Date;
}

class WebhookEndpointService {
  async createEndpoint(
    tenantId: string,
    config: CreateEndpointRequest,
  ): Promise<WebhookEndpoint> {
    const endpoint: WebhookEndpoint = {
      id: crypto.randomUUID(),
      tenantId,
      url: config.url,
      description: config.description,
      secret: crypto.randomBytes(32).toString('hex'),
      subscribedEvents: config.subscribedEvents,
      enabled: true,
      retryConfig: {
        maxAttempts: config.maxRetries || 5,
        baseDelayMs: config.baseDelayMs || 1000,
      },
      rateLimit: {
        requestsPerMinute: config.requestsPerMinute || 100,
        currentWindow: 0,
        windowCount: 0,
      },
      headers: config.headers,
      createdAt: new Date(),
      updatedAt: new Date(),
    };

    // Validate URL
    this.validateUrl(endpoint.url);

    // Validate event types
    this.validateEventTypes(endpoint.subscribedEvents);

    // Test endpoint
    if (config.testOnCreate) {
      await this.sendTestEvent(endpoint);
    }

    await this.store.create('webhook_endpoints', endpoint);
    await this.invalidateCache(endpoint.id);

    return endpoint;
  }

  async updateEndpoint(
    id: string,
    tenantId: string,
    updates: Partial<WebhookEndpoint>,
  ): Promise<WebhookEndpoint> {
    const endpoint = await this.getEndpoint(id, tenantId);
    if (!endpoint) throw new Error('Endpoint not found');

    const updated = {
      ...endpoint,
      ...updates,
      updatedAt: new Date(),
    };

    // If URL changed, validate
    if (updates.url) {
      this.validateUrl(updates.url);
    }

    // If events changed, validate
    if (updates.subscribedEvents) {
      this.validateEventTypes(updates.subscribedEvents);
    }

    await this.store.update('webhook_endpoints', { id }, updated);
    await this.invalidateCache(id);

    return updated;
  }

  async rotateSecret(id: string, tenantId: string): Promise<{ newSecret: string }> {
    const endpoint = await this.getEndpoint(id, tenantId);
    const newSecret = crypto.randomBytes(32).toString('hex');

    // Move current secret to secondary
    await this.store.update('webhook_endpoints', { id }, {
      secondarySecret: endpoint.secret,
      secret: newSecret,
      updatedAt: new Date(),
    });

    return { newSecret };
  }

  async sendTestEvent(endpoint: WebhookEndpoint): Promise<boolean> {
    const testEvent = {
      id: `test-${crypto.randomUUID()}`,
      type: 'webhook.test',
      payload: { message: 'This is a test webhook from Voice Agent' },
      idempotencyKey: `test-${Date.now()}`,
      timestamp: new Date().toISOString(),
      tenantId: endpoint.tenantId,
    };

    try {
      const response = await fetch(endpoint.url, {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json',
          'X-Webhook-ID': testEvent.id,
          'X-Webhook-Test': 'true',
        },
        body: JSON.stringify(testEvent),
        signal: AbortSignal.timeout(5000),
      });

      return response.ok;
    } catch {
      return false;
    }
  }

  async checkRateLimit(endpoint: WebhookEndpoint): Promise<boolean> {
    const now = Math.floor(Date.now() / 60000); // Current minute window
    const { rateLimit } = endpoint;

    if (rateLimit.currentWindow !== now) {
      // Reset window
      rateLimit.currentWindow = now;
      rateLimit.windowCount = 0;
    }

    rateLimit.windowCount++;
    return rateLimit.windowCount <= rateLimit.requestsPerMinute;
  }

  private validateUrl(url: string): void {
    try {
      const parsed = new URL(url);
      if (!['http:', 'https:'].includes(parsed.protocol)) {
        throw new Error('URL must use HTTP or HTTPS protocol');
      }
    } catch (error) {
      throw new Error(`Invalid webhook URL: ${error.message}`);
    }
  }

  private validateEventTypes(events: string[]): void {
    const validTypes = [
      'agent.created', 'agent.updated', 'agent.deleted',
      'call.completed', 'call.failed', 'call.recording.ready',
      'alert.fired', 'alert.resolved',
    ];

    for (const event of events) {
      if (!validTypes.includes(event)) {
        throw new Error(`Invalid event type: ${event}`);
      }
    }
  }
}
```

## Integration Points

- **Developer Portal**: Configuration UI for endpoint management
- **Delivery Engine**: Reads endpoint configuration for routing
- **Secret Store**: Encrypted secret storage in database

## Production Considerations

- **URL Validation**: Block private/internal IP ranges
- **Secret Strength**: Enforce minimum 32-byte secrets
- **Endpoint Limits**: Maximum 10 endpoints per tenant
- **Replay Protection**: Test events excluded from replay attack detection

## Open-Source Tools

- **Zod**: Request validation for configuration API
- **tRPC**: Type-safe API for portal backend
