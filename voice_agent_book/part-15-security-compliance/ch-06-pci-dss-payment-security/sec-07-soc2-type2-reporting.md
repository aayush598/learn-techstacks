# Section 07: SOC 2 Type II Reporting

SOC 2 Type II reports provide assurance that controls operated effectively over a period (typically 6-12 months). The report includes the auditor's opinion, management's assertion, control descriptions, test results, and any exceptions. The platform provides the SOC 2 report to enterprise prospects and existing customers under NDA.

Report timeline: 2 months preparation (evidence aggregation, pre-audit readiness assessment) → 2 weeks on-site audit (control testing, interviews, evidence review) → 2 weeks report drafting → 1 week review and sign-off. Total: approximately 6 months for Type I (point-in-time), 12 months for Type II (period-of-time).

The platform maintains a SOC 2 report portal where authorized tenants can download the latest report (under NDA). Report notifications are sent when new reports are available. The SOC 2 report covers all five trust principles. Bridge letters are provided between report periods to confirm no significant control changes.
