# Performance Tracker Template

## Problem Tracking Spreadsheet

Create a spreadsheet (Google Sheets / Excel / Notion) with these columns:

| Column | Example | Purpose |
|--------|---------|---------|
| Date | 2026-01-01 | When you solved it |
| Problem | Two Sum | Problem name |
| Link | LeetCode 1 | Link to problem |
| Topic | Arrays | Main topic |
| Pattern | HashMap | Algorithm pattern |
| Difficulty | Easy | Easy/Medium/Hard |
| Time (min) | 12 | How long it took |
| Attempts | 2 | Times tried |
| Status | Solved | Solved/Revise/Stuck |
| Notes | HashMap for complement | Key learning |
| Next Review | 2026-01-02 | Next review date |
| Review Result | Easy | Easy/Hard/NoRecall |

## Conditional Formatting Rules

- "NoRecall" → Red (immediate re-review needed)
- "Hard" → Yellow (review within 3 days)
- "Easy" → Green (review on schedule)
- "Revise" → Orange (re-solve within 1 day)
