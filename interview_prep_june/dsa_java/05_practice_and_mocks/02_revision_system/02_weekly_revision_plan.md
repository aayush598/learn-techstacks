# Weekly Revision Plan

## 12-Week Interview Prep Schedule

### Weeks 1-2: Foundations + Arrays/Strings
```
Mon-Tue: Arrays basics, two pointers, kadane
Wed-Thu: Strings, anagram, palindrome patterns
Fri-Sat: Prefix sum, sliding window
Sun: Full review + 30 problems from this week
```

### Weeks 3-4: Linked List + Stack + Queue
```
Mon-Tue: Singly LL implementation, reversal, cycle
Wed-Thu: Stack patterns (monotonic, NGE, histogram)
Fri-Sat: Queue patterns (deque, sliding max)
Sun: Review + 30 problems
```

### Week 5: Recursion + Hashing
```
Mon-Tue: Recursion patterns (subsets, permutations)
Wed-Thu: HashMap/HashSet problems
Fri-Sat: LRU cache, grouping, frequency problems
Sun: Review
```

### Week 6: Trees
```
Mon-Tue: Binary tree traversals (DFS + BFS)
Wed-Thu: BST operations, validation
Fri-Sat: Tree problems (LCA, views, path sum)
Sun: Review + 20 tree problems
```

### Week 7: Graphs
```
Mon-Tue: BFS/DFS, connected components, bipartite
Wed-Thu: Shortest path (Dijkstra, Bellman-Ford)
Fri-Sat: Topological sort, union-find
Sun: Review + 20 graph problems
```

### Week 8: Heaps + Tries + Advanced DS
```
Mon-Tue: Heap patterns (Top K, median in stream)
Wed-Thu: Trie (autocomplete, word search)
Fri-Sat: Segment tree, Fenwick tree, DSU
Sun: Review
```

### Week 9: Sorting + Searching
```
Mon-Tue: Quick sort, merge sort, counting sort
Wed-Thu: Binary search variants, search on answer
Fri-Sat: Two-pointer, sliding window problems
Sun: Review
```

### Week 10: DP
```
Mon-Tue: 1D DP (fib, house robber, LIS)
Wed-Thu: 2D DP (knapsack, LCS, edit distance)
Fri-Sat: DP on grids, strings, trees
Sun: Review + 25 DP problems
```

### Week 11: Backtracking + Greedy + Bit
```
Mon-Tue: Backtracking (subsets, permutations, N-Queens)
Wed-Thu: Greedy (activity selection, scheduling)
Fri-Sat: Bit manipulation basics + problems
Sun: Review
```

### Week 12: Company-wise + Mock Interviews
```
Mon-Tue: Google/Amazon focused problems
Wed-Thu: Meta/Microsoft/Apple focused
Fri: Full mock interview (45 min)
Sat: Review weak areas
Sun: Rest + final prep
```

## Daily Breakdown

| Time | Activity |
|------|----------|
| 30 min | Review yesterday's problems |
| 45 min | Study new concept + examples |
| 45 min | Solve 3-5 new problems |
| 15 min | Log progress, schedule reviews |
| **Total** | **2.25 hours/day** |
