## 🧠 Mental Model: The DSA Decision Tree

When you see any problem, run through this tree:

```
PROBLEM
├─ Is there a linear collection? → Arrays / Strings / LinkedList
├─ Need efficient lookups? → HashMap / HashSet / TreeMap
├─ Need order maintained? → Stack / Queue / Deque / Heap
├─ Hierarchical data? → Tree / Trie
├─ Relationships between entities? → Graph
└─ Need optimization (min/max/count)? →
    ├─ Optimal substructure? → DP
    ├─ Greedy works? → Greedy
    ├─ Explore all? → Backtracking / Recursion
    └─ Narrow down search space? → Binary Search
```
