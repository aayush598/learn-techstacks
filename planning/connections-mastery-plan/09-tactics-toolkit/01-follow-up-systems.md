# Follow-Up Systems

## The Art of Persistent, Respectful Follow-Up

### Why Follow-Up Separates Masters from Amateurs

80% of connections are made on the follow-up, not the first contact. Most people send one message and give up. You will not be most people.

The follow-up shows:
- You're serious (not just casting a wide net)
- You're respectful (not pushy, but persistent)
- You're organized (you track and remember)
- You care (you genuinely want to connect)

### The Follow-Up Cadence

**Cold Outreach Sequence (if no reply):**

```
Message 1 (Day 0): Initial outreach
Message 2 (Day 5-7): Follow-up with new value
Message 3 (Day 14-21): Final follow-up, different angle
→ If no reply after 3 attempts, park for 3 months, then retry
```

**Template 2 - New Value Follow-Up (Day 5-7):**
```
"Hi [Name], following up on my previous message. I also came across [new article/resource/insight] that I thought might be relevant to your work on [topic]. Would love to hear your thoughts if you have a moment."
```

**Template 3 - Different Angle Follow-Up (Day 14-21):**
```
"Hi [Name], I know you're busy so I'll keep this brief. I'm [specific context] and I'd really value your perspective on [specific question]. Even a one-line reply would be incredibly helpful."
```

### The Hold-All System

For each connection, track these dates and don't let any slip:

```
Last Contacted: [Date]
Follow-Up 1 Due: [Date]
Follow-Up 2 Due: [Date]
Dormant Alert: [Date - 90 days no contact]
Re-engagement Due: [Date]
```

### The 24-Hour Rule

For EVERY response you receive, reply within 24 hours. This signals:
- You're responsive and reliable
- You value them
- You're organized

Even if you can't fully answer: "Got your message, will reply properly within 48 hours."

### The Weekly Follow-Up Batch

Every Friday, batch-process all follow-ups:
1. Check CRM for pending follow-ups
2. Send all follow-up messages (copy-paste from templates, customize slightly)
3. Log dates in CRM
4. Move non-responders to "parked" after 3 attempts

### The Re-Engagement Sequence (for dormant connections)

**If 90+ days since last contact:**

```
Step 1: Like their recent content (low effort)
Step 2: If they engage, send DM: "Hey [Name], it's been a while! Hope you're doing well. I was thinking about [reference to your last conversation] and wanted to check in."
Step 3: If no engagement after step 1, send DM: "Hi [Name], it's been a while — hope all is well. I came across [something relevant] and thought of you. How have you been?"
```

### Follow-Up Automation Tools

**Manual (up to 200 connections):**
- Use CRM with date reminders
- Set calendar reminders for follow-ups
- Batch process weekly

**Semi-Automated (200-1000 connections):**
- Use LinkedIn automation tools carefully
- Set up email sequences (Mailchimp, HubSpot free tier)
- Use CRM workflows

**Note:** Never fully automate personal follow-ups. Personalization is the entire point.

### Common Follow-Up Mistakes

| Mistake | Why It's Bad | Better Approach |
|---------|-------------|-----------------|
| Following up too fast (<3 days) | Seems desperate | Wait 5-7 days |
| Using the same message | Shows no effort | Add new value each time |
| Guilt-tripping ("I know you're busy but...") | Creates negative feeling | Simply be respectful |
| Asking "Did you get my message?" | Passive-aggressive | Provide new value |
| More than 3 follow-ups | Becomes harassment | Park and retry later |
