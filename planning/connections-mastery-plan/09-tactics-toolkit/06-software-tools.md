### Tool Integration Workflow

```
Inbound (LinkedIn/Twitter/Email)
        ↓
CRM (Notion/Airtable) — logs every connection
        ↓
Actions (Calendly for meetings, Grammarly for writing)
        ↓
Follow-ups (CRM reminders, email templates)
        ↓
Notes (Obsidian for knowledge capture)
        ↓
Content (Canva for graphics, Buffer for scheduling)
        ↓
Analytics (LinkedIn/Twitter built-in analytics)
```

### Automation Rules

**If you use automation, follow these ethics:**
1. Never automate messages (personalization is the entire point)
2. Use automation only for: reminders, logging, scheduling, analytics
3. Stay within platform limits (LinkedIn: ~100 actions/week)
4. Never scrape data without permission
5. Review all automated actions regularly

### Tool Stack by Phase

**Phase 1 (0-200 connections):**
- Google Sheets (CRM)
- Calendly (scheduling)
- Grammarly (writing)
- Canva (content)
- Free version of everything

**Phase 2 (200-1000 connections):**
- Notion (CRM)
- LinkedIn Premium
- Buffer (content scheduling)
- Otter.ai (call notes)

**Phase 3 (1000-5000 connections):**
- Airtable or Dex (CRM)
- Apollo.io (email finding)
- Zapier (automation)
- Descript (content creation)

**Phase 4 (5000+ connections):**
- Clay (advanced CRM)
- Custom-built tools
- Virtual assistant for maintenance
- Full automation stack
