# Weekly Plan

## The Connector's Week

### Weekly Rhythm

| Day | Theme | Focus |
|-----|-------|-------|
| Monday | Plan + Create | Plan week, create 2 content pieces |
| Tuesday | Outreach + Engage | Send 15 connection requests, engage 20 people |
| Wednesday | Deepen + Calls | Have 2-3 calls, follow up on key connections |
| Thursday | Content + Community | Publish, attend 1 event |
| Friday | Follow-up + Logistics | Follow up on all unanswered, CRM cleanup |
| Saturday | Learning + Review | Read, learn, review weekly metrics |
| Sunday | Rest + Plan next week | Recharge, plan |

### Weekly Targets (Phase 1-2)

**Connections:**
- New connections: 10-15
- DMs sent: 25-35
- Calls/meetings: 3-5
- Follow-ups sent: 10-15

**Content:**
- Posts published: 3-5
- Comments left: 20-30
- Content pieces created: 2-3

**Learning:**
- Articles read: 5-10
- Book chapters read: 5-7
- Domain deep-dives: 1-2

**System:**
- CRM updates: 15-20
- Weekly review completed: Yes/No

### Weekly Targets (Phase 3-5)

**Connections:**
- Strategic introductions: 2-3
- High-value meetings: 3-5
- Alliance deepening actions: 5-10

**Content:**
- Thought leadership pieces: 2-3
- Speaking engagements: 0-1
- Media interactions: 1-2

**System:**
- Network health check: Done
- Value delivered analysis: Done
- Next week strategy: Planned

### Sunday Planning Protocol

**30 minutes every Sunday:**

1. **Review last week (10 min):**
```
Connections made: ___/target
Calls held: ___/target
Content published: ___/target
Value delivered: [Top 3]
Biggest win: ___
Biggest lesson: ___
```

2. **Plan next week (15 min):**
```
Target connections for week: [List 10-15]
Calls to schedule: [List 3-5]
Content to publish: [Schedule]
Knowledge focus: [1 domain]
Events to attend: [List]
```

3. **Set weekly intention (5 min):**
```
This week, I will focus on: [1 theme]
The most important connection to make: [Name]
The most important thing to learn: [Topic]
```
