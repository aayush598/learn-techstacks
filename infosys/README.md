## Strategy to Crack SP L3

1. **Q1 (Easy - 20 marks)** - Solve in 30 min. Usually arrays/strings. Must get 100%.
2. **Q2 (Medium - 30 marks)** - Solve in 60 min. Greedy or Graph. Target 100%.
3. **Q3 (Hard - 50 marks)** - Solve in 90 min. DP or Complex Graph. Target 70-100%.

**Total target: 80-100% to get L3 call.**

## Tips

- Write clean, modular Python code
- Handle edge cases (empty input, single element, large values)
- Use meaningful variable names
- Comment key logic (but don't over-comment)
- Test with sample cases before submitting
- If stuck on Q3, attempt brute force for partial marks
