# TypeScript Mastery - From Zero to FAANG Level

A comprehensive, production-grade TypeScript reference covering everything from fundamentals to advanced type manipulation, design patterns, and interview preparation for top tech companies.

## Directory Structure

| Folder | Description |
|--------|-------------|
| `01-fundamentals/` | TypeScript basics, setup, primitives, variables, type inference |
| `02-types/` | Complete type system: primitives, arrays, tuples, enums, unions, intersections |
| `03-functions/` | Function types, overloading, generics in functions, closures |
| `04-classes-and-oop/` | OOP in TypeScript: classes, inheritance, polymorphism, mixins |
| `05-generics/` | Generics deep dive: constraints, defaults, conditional types, mapped types |
| `06-advanced-types/` | Type manipulation: keyof, typeof, infer, recursive types, branded types |
| `07-modules-and-namespaces/` | ES modules, CommonJS, module resolution, path mapping |
| `08-decorators/` | Class/method/property decorators, decorator factories, NestJS patterns |
| `09-error-handling/` | Custom errors, Result type pattern, exhaustive error handling |
| `10-promises-and-async/` | Promises, async/await, generators, iterators, async generators |
| `11-design-patterns/` | GoF patterns in TypeScript: Singleton, Factory, Observer, Strategy, etc. |
| `12-react-with-typescript/` | React + TS: components, hooks, context, forms, advanced patterns |
| `13-nodejs-with-typescript/` | Node.js + TS: Express, Prisma, GraphQL, WebSockets |
| `14-testing/` | Jest, unit/integration testing, mocking, type testing |
| `15-performance-and-optimization/` | Compilation, tree-shaking, declaration files, DefinitelyTyped |
| `16-typescript-compiler/` | Compiler API, transformers, tsconfig deep dive, project references |
| `17-interview-prep/` | Coding challenges, system design, company-specific questions |
| `18-real-world-patterns/` | API clients, state management, DI, plugin architecture, event-driven |

## How to Use

1. Start with `01-fundamentals/` if you're new to TypeScript
2. Skip to specific topics using the directory structure
3. Each folder ends with `interview-questions.md` containing FAANG-level Q&A
4. Check `17-interview-prep/` for comprehensive interview preparation

## Prerequisites

- JavaScript ES6+ knowledge
- Node.js installed
- A code editor (VS Code recommended)

## Quick Start

```bash
npm install -g typescript
tsc --init
```
